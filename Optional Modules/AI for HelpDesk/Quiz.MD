## **Section 1: Multiple Choice (Choose the best answer)**

**1. Which task is AI best suited for in helpdesk operations?**
a) Resolving a complex server outage
b) Classifying incoming tickets
c) Handling security policy violations
d) Negotiating SLAs with clients

**Answer:** b) Classifying incoming tickets

---

**2. AI can assist helpdesk agents by:**
a) Routing tickets to the correct team
b) Making ethical decisions independently
c) Fully replacing human agents
d) Handling high-stakes escalations

**Answer:** a) Routing tickets to the correct team

---

**3. Which of these is NOT a common AI application in helpdesk?**
a) Auto-suggesting solutions from knowledge bases
b) Summarizing long chat transcripts
c) Performing emotional conflict resolution
d) Predictive system monitoring

**Answer:** c) Performing emotional conflict resolution

---

**4. When should a human agent intervene instead of AI?**
a) Routine password resets
b) Escalated technical issues or sensitive incidents
c) FAQ responses
d) Automated ticket summaries

**Answer:** b) Escalated technical issues or sensitive incidents

---

**5. Benefits of using AI in helpdesk include:**
a) Faster ticket resolution
b) Consistent responses
c) Reduced agent workload
d) All of the above

**Answer:** d) All of the above
