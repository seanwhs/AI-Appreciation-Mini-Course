## **Section 3 ‚Äî Short Answer (5 Questions)**

1. Name two AI tools for code autocompletion and boilerplate generation.
   **Sample Answer:** GitHub Copilot, Replit Ghostwriter

2. How does AI assist in debugging and error detection?
   **Sample Answer:** Highlights syntax and logical errors, suggests fixes, provides explanations, and recommends tests

3. Give one example of how AI improves code readability or performance.
   **Sample Answer:** Suggests refactoring patterns, removes duplicate code, optimizes loops or data structures

4. How can AI assist in learning or exploring new codebases?
   **Sample Answer:** Generates example code, explains logic line-by-line, supports interactive experimentation (Vibe Coding)

5. Why is human oversight still essential when using AI for coding?
   **Sample Answer:** Developers handle architecture, design decisions, ethical concerns, complex logic, and creative problem-solving

## **Section 4 ‚Äî Application / Scenario (3 Questions)**

1. You are tasked with creating a CRUD API in Python. How can AI assist?
   **Sample Answer:** Suggests boilerplate endpoints, generates function skeletons, adds docstrings, and proposes input validation

2. A legacy codebase has failing test cases. What AI-assisted steps could you take?
   **Sample Answer:** Detect syntax/logical errors, suggest fixes, generate missing unit tests, provide explanations for errors

3. Your team wants to improve pull request reviews. How can AI help?
   **Sample Answer:** Highlights potential bugs, suggests readability or security improvements, generates review summaries or inline comments
