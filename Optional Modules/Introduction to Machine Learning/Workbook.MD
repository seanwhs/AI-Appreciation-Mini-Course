# **Machine Learning Workbook**

## **Section 1: Introduction to Machine Learning**

### **1.1 What is Machine Learning?**

**Definition:** Machine Learning (ML) is a type of AI that allows computers to **learn patterns from data** and make predictions or decisions without being explicitly programmed.

**Reflection Questions:**

1. Think of an example from your daily life where a computer seems to “learn” from data. Describe it.
2. How is ML different from traditional programming?

---

### **1.2 Types of Machine Learning**

| Type                   | Description                                 | Example                     |
| ---------------------- | ------------------------------------------- | --------------------------- |
| Supervised Learning    | Learns from labeled data (input + output)   | Predicting house prices     |
| Unsupervised Learning  | Finds patterns in unlabeled data            | Customer segmentation       |
| Reinforcement Learning | Learns through trial and error with rewards | AI playing games like chess |

**Activity:**

* Identify one example of each type of ML in real life.

---

## **Section 2: Key Concepts in Machine Learning**

### **2.1 Core Concepts**

* **Data** – Raw information used to train models
* **Features** – Attributes or variables used for prediction
* **Model** – Algorithm that learns from data
* **Training** – Teaching the model using data
* **Evaluation** – Measuring model performance

**Exercise:**

1. Imagine you are predicting the price of a car. List 5 features you would use.
2. What would be the “output” in this scenario?

---

### **2.2 ML Workflow**

1. Collect Data
2. Explore & Clean Data
3. Choose a Model
4. Train the Model
5. Evaluate Model Performance
6. Deploy or Use Model

**Reflection:**

* Why do you think data cleaning is important before training a model?

## **Section 4: Hands-On Activities**

### **Activity 1: Simple Classification with Iris Dataset**

**Goal:** Predict the type of iris flower based on measurements.

**Steps:**

1. Open Google Colab or Jupyter Notebook.
2. Load the Iris dataset.
3. Visualize the features (e.g., sepal length vs petal length).
4. Train a simple model (e.g., Decision Tree or Logistic Regression).
5. Test the model with a new data point.

**Exercise:**

* Record your prediction results.
* Reflect: What features were most important in predicting the flower type?

---

### **Activity 2: Image Recognition with Teachable Machine**

**Goal:** Train a model to classify images.

**Steps:**

1. Go to Teachable Machine.
2. Upload images for at least two categories (e.g., apples vs bananas).
3. Train the model.
4. Test it with new images.

**Exercise:**

* How accurate was your model?
* How could you improve it?

---

### **Activity 3: Text Analysis with NLP (Optional)**

**Goal:** Analyze text for sentiment.

**Steps:**

1. Use Python with NLTK or spaCy.
2. Load a small dataset of positive and negative reviews.
3. Train a model to classify sentiment.
4. Test with new sentences.

**Exercise:**

* Record your observations: Which words were most important for predictions?

## **Section 6: Mini Project**

**Goal:** Apply ML to a simple real-world problem.

**Steps:**

## **Section 7: Summary and Reflection**

**Key Takeaways:**

* Machine Learning allows computers to learn patterns from data.
* There are different types of ML for different problems.
* Applications of ML range from predictions to image recognition and recommendations.
* Hands-on experimentation helps reinforce concepts.
* Ethical considerations are critical for responsible ML use.
