## **Quiz: Artificial Intelligence (AI) and Its Subfields**

---

### **1. Multiple-Choice Questions (MCQs)**

1. **Which of the following is the best definition of Artificial Intelligence (AI)?**
   a) A system that can perform only basic calculations
   b) Machines that can perform tasks that normally require human intelligence
   c) Machines that perform tasks only when programmed with explicit instructions
   d) A set of pre-defined rules for making decisions

2. **Which of the following best describes **General AI**?**
   a) AI that performs a single task, like a recommendation system
   b) AI that can perform any intellectual task a human can, but it is still hypothetical
   c) AI that surpasses human intelligence in all fields
   d) AI that can generate creative content like images and text

3. **What is the key feature of **Agentic AI**?**
   a) It works only with pre-programmed instructions.
   b) It can plan, act, and adapt independently to achieve goals.
   c) It is only used for creating content like text or images.
   d) It is focused solely on learning from data.

4. **Which of the following is an example of **Generative AI**?**
   a) A chatbot that answers customer queries based on a set of rules
   b) A system that generates human-like text based on previous input, such as GPT-3
   c) A recommendation system for movies or products
   d) A machine learning model for predicting stock prices

5. **Which of the following is a typical application of **Deep Learning (DL)**?**
   a) Image recognition through Convolutional Neural Networks (CNNs)
   b) Predicting house prices using linear regression
   c) Customer segmentation using unsupervised learning
   d) Game-playing AI using Reinforcement Learning

6. **In which scenario would **Reinforcement Learning** be the most applicable?**
   a) Predicting the weather based on historical data
   b) Detecting fraud in financial transactions
   c) Training a robot to play chess through trial and error
   d) Segmenting customers into groups based on behavior

### **5. Scenario-Based Questions**

16. **Scenario:** A healthcare provider uses an AI system to analyze medical images for early detection of diseases. The system is trained on labeled data of healthy and unhealthy scans. What type of Machine Learning (ML) is this system most likely using? Explain why.

17. **Scenario:** You are developing a chatbot using **Generative AI** to assist customers. The chatbot can generate responses to customer queries, but sometimes the responses are incorrect or nonsensical. How could this issue be addressed or improved with better training or oversight?

    d) Machine Learning

### **Explanation of Correct Answers:**

1. AI refers to machines that perform human-like tasks, such as decision-making, recognizing patterns, or understanding language.
2. General AI is a hypothetical type of AI that would perform all tasks a human can, whereas Narrow AI specializes in specific tasks.
