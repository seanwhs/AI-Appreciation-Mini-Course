# **SECTION 2 ‚Äî MACHINE LEARNING (ML)**

## **2.1 What is Machine Learning?**

ML is a subset of AI where machines **learn patterns from data** instead of being explicitly programmed.

### **Three Types of ML:**

| Type                       | What it Does              | Example               |
| -------------------------- | ------------------------- | --------------------- |
| **Supervised Learning**    | Learns from labeled data  | Email spam detection  |
| **Unsupervised Learning**  | Finds hidden patterns     | Customer segmentation |
| **Reinforcement Learning** | Learns by trial and error | Game-playing agents   |

# **SECTION 3 ‚Äî DEEP LEARNING (DL)**

## **3.1 What is Deep Learning?**

Deep Learning is a subset of ML that uses **neural networks with many layers** to learn complex patterns.

### **Neural Network Diagram (Simplified)**

```
Input Layer ‚Üí Hidden Layer 1 ‚Üí Hidden Layer 2 ‚Üí Output Layer
       ‚óè ‚óè ‚óè         ‚óè ‚óè ‚óè           ‚óè ‚óè ‚óè            ‚óè ‚óè
```

---

## **3.2 Network Types**

| Network Type                           | Best For                   | Examples             |
| -------------------------------------- | -------------------------- | -------------------- |
| **CNN (Convolutional Neural Network)** | Images                     | Face recognition     |
| **RNN / LSTM**                         | Sequences                  | Language translation |
| **Transformers**                       | Long sequences + reasoning | ChatGPT, BERT        |

# **SECTION 5 ‚Äî AGENTIC AI**

## **5.1 What is Agentic AI?**

Agentic AI systems can **act autonomously**, plan, and make decisions to achieve goals.

Unlike standard AI (reacts to prompts), agentic AI can:

* set or refine goals
* choose actions
* adapt to changing environments

---
